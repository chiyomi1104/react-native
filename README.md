# Nucamp Site Mobile App
## Overview
This is a mobile app for Nucamp coding school. I made this app as a campstone for React Native course. 

## Development
### Languages and Framework
- React Native
- Expo

### Features
- Login/Logout
- Likes and comments
- Swipe
- Create, read, update, delete and store comments
- Chage user picture
- OAuth
